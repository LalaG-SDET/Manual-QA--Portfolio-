
# Manual Testing Project: Tap.az Platform Analysis Az

### Objective
Comprehensive manual testing of the Tap.az platform, focusing on the "Post Ad" and "Search Filter" modules to ensure system reliability and user experience.

### Testing Techniques Applied
- **Boundary Value Analysis (BVA):** Precision testing on price and year filters.
- **Equivalence Partitioning (EP):** Validating category selections and user input fields.
- **UI/UX Testing:** Verified responsiveness across mobile and desktop interfaces.

### Key Deliverables
- **Test Cases:** 50+ detailed scenarios for functional validation.
- **Bug Reports:** Real-world defects identified with severity levels and reproduction steps.
